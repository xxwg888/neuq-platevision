# Notes: 车牌识别课程设计

## 系统定位
- 主线符合数字图像处理课程设计：OpenCV 传统算法可解释、可展示中间过程。
- 工程扩展体现完整系统：前后端分离、批量评测、远程推理 provider。
- 后续服务器模型完成后，本地只替换 provider，不需要重写前端。

## 数据集规划
- CCPD: 蓝牌/绿牌主实验与鲁棒性分析。
- CRPD: 黄牌、白牌、双层牌、多车牌扩展。
- CBLPRD-330k: 合成多类型车牌，用于字符识别增强。

## UI 方向
- 风格：科研控制台，强调数据、过程和可视化证据。
- 第一屏直接是可用工作台，不做营销式落地页。
- 必须展示原图、检测结果、裁剪车牌、二值图、字符切分、识别结果和耗时。
